## 15. f-strings
#### 💻 ターミナルで実行
<div align="right">
  <a href="../README.md#section3">◀️READMEに戻る</a>

```bash
# f-strings
>>> a = 'a'
>>> f'a is {a}'
'a is a'
>>> x, y, z = 1, 2, 3
>>> f'a is {x}, {y}, {z}'
'a is 1, 2, 3'
```
