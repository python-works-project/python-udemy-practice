---
### () の使い所
- タプルの定義 : `(1, 2, 3)`
- 1要素タプルはカンマ必須 : `(1,)`
- 関数呼び出し : `print("Hello")`
- ジェネレータ式 : `(x**2 for x in range(5))`
- 演算のグルーピング : `(2 + 3) * 4`
- 関数定義の引数 : `def func(a, b): ...`

---

### [] の使い所
- リストの定義 : `[1, 2, 3]`
- インデックスアクセス : `lst[0]`
- スライス : `lst[1:3]`, `lst[::2]`
- 辞書のキーアクセス : `d['x']`
- リスト内包表記 : `[x**2 for x in range(5)]`
- 型ヒント (Python 3.9以降) : `list[int]`, `dict[str, int]`
- 多次元リストのアクセス : `matrix[0][1]`

---

### {} の使い所
- 辞書の定義 : `{'x': 10, 'y': 20}`
- 集合 (set) の定義 : `{'a', 'b', 'c'}`
- 辞書内包表記 : `{k: v for k, v in pairs}`
- 集合内包表記 : `{x**2 for x in range(5)}`
- 辞書の更新・追加 : `d['z'] = 30`

---

### ✅ ポイントまとめ
- `[]` → リスト、インデックス、スライス、辞書キー
- `()` → タプル、関数呼び出し、演算グルーピング
- `{}` → 辞書、集合、内包表記
