# Python_study

このリポジトリは、Udemy講座  
**「現役シリコンバレーエンジニアが教えるPython 3 入門 + 応用 + アメリカのシリコンバレー流コードスタイル」**  
で学んだ内容を整理した学習ログです。

## 構成 (Structure)
- basics/ : Python基礎（変数・演算子・制御構文）
- functions_classes/ : 関数・クラス・オブジェクト指向
- data_structures/ : データ構造（リスト・辞書・集合・タプル）
- modules_packages/ : モジュール・パッケージ・importの使い方
- file_io/ : ファイル入出力・例外処理
- advanced_topics/ : 応用編（デコレーター・ジェネレーター・イテレーター）
- web_scraping/ : Webスクレイピング（requests, BeautifulSoup）
- data_analysis/ : データ分析（NumPy, Pandas, Matplotlib）
- silicon_valley_style/ : シリコンバレー流コードスタイル（PEP8, ベストプラクティス）

## 環境 (Environment)
- Python 3.13.5 (Anaconda)
- Visual Studio Code

## 目的 (Purpose)
- 学習の進捗管理
- ポートフォリオとして活用
