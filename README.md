このリポジトリは、Udemy講座や書籍で学んだPythonコードを整理した学習ログです。  
基礎からデータ分析まで、学習の進捗を記録しています。

## 構成
- basics/ : Python基礎（変数・ループ・関数）
- web_scraping/ : Webスクレイピングの練習コード
- data_analysis/ : PandasやMatplotlibの練習コード

## 環境
