```bash
# タプルの基本
t = (1,2,3,4,1,2)
type(t)             # <class 'tuple'> → タプル型
t                   # (1, 2, 3, 4, 1, 2)

# タプルはイミュータブル（変更不可）
t[0] = 100          # エラー: 'tuple' object does not support item assignment

# スライスやインデックス参照は可能
t[2:5]              # (3, 4, 1)
t[2]                # 3

# index() → 指定値の位置を返す
t.index(1)          # 0（最初の1の位置）
t.index(1,1)        # 4（インデックス1以降で最初に出る1の位置）

# count() → 指定値の出現回数
t.count(1)          # 2

# タプルの中にリストを入れると、リスト部分は変更可能
t = ([1,2,3], [4,5,6])
t                   # ([1, 2, 3], [4, 5, 6])
t[0] = [1]          # エラー: タプル自体の要素の入れ替えは不可
t[0][0]             # 1（リストの要素は参照できる）
t[0][0] = 100       # リストの要素は変更可能
t                   # ([100, 2, 3], [4, 5, 6])

# タプルの作り方いろいろ
t = 1,2,3           # (1, 2, 3) → カンマ区切りでタプル
type(t)             # <class 'tuple'>

t = 1,              # (1,) → 要素1つのタプルはカンマ必須
type(t)             # <class 'tuple'>

t = 1               # これは単なるint
type(t)             # <class 'int'>

t = ()              # 空タプル
type(t)             # <class 'tuple'>

t = (1)             # これはint（カンマがないため）
type(t)             # <class 'int'>

# タプルの結合例：+ 演算子で新しいタプルを作成できる
new_tuple = (1,2,3) + (4,5,6)   # タプル同士の結合
new_tuple                        # (1, 2, 3, 4, 5, 6)

# (1) は int なので + (4,5,6) はエラー
# new_tuple = (1) + (4,5,6) → TypeError

# (1,) はタプルなので結合可能
new_tuple = (1,) + (4,5,6)
new_tuple                        # (1, 4, 5, 6)
```

## 📝 学習のまとめ
- タプルはリストと違って値の代入ができない
- 後で書き換えられたくないもの、読み込み専用で使用
- t = 1,　の様にカンマを付けるとタプルとして扱われる
