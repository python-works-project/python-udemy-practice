```bash
# タプルの基本
t = (1,2,3,4,1,2)
type(t)             # <class 'tuple'> → タプル型
t                   # (1, 2, 3, 4, 1, 2)

# タプルはイミュータブル（変更不可）
t[0] = 100          # エラー: 'tuple' object does not support item assignment

# スライスやインデックス参照は可能
t[2:5]              # (3, 4, 1)
t[2]                # 3

# index() → 指定値の位置を返す
t.index(1)          # 0（最初の1の位置）
t.index(1,1)        # 4（インデックス1以降で最初に出る1の位置）

# count() → 指定値の出現回数
t.count(1)          # 2

# タプルの中にリストを入れると、リスト部分は変更可能
t = ([1,2,3], [4,5,6])
t                   # ([1, 2, 3], [4, 5, 6])
t[0] = [1]          # エラー: タプル自体の要素の入れ替えは不可
t[0][0]             # 1（リストの要素は参照できる）
t[0][0] = 100       # リストの要素は変更可能
t                   # ([100, 2, 3], [4, 5, 6])

# タプルの作り方いろいろ
t = 1,2,3           # (1, 2, 3) → カンマ区切りでタプル
type(t)             # <class 'tuple'>

t = 1,              # (1,) → 要素1つのタプルはカンマ必須
type(t)             # <class 'tuple'>

t = 1               # これは単なるint
type(t)             # <class 'int'>

t = ()              # 空タプル
type(t)             # <class 'tuple'>

t = (1)             # これはint（カンマがないため）
type(t)             # <class 'int'>

# タプルの結合例：+ 演算子で新しいタプルを作成できる
new_tuple = (1,2,3) + (4,5,6)   # タプル同士の結合
new_tuple                        # (1, 2, 3, 4, 5, 6)

# (1) は int なので + (4,5,6) はエラー
# new_tuple = (1) + (4,5,6) → TypeError

# (1,) はタプルなので結合可能
new_tuple = (1,) + (4,5,6)
new_tuple                        # (1, 4, 5, 6)
```

## 📝 学習のまとめ
- 丸括弧 () を使う
```bash
t = (1, 2, 3)
```
- カンマ区切りだけでもタプルになる
```bash
t = 1, 2, 3    # (1, 2, 3)
```
- 1要素タプルはカンマ必須
```bash
t = (1,)       # (1,)
t = 1,         # (1,)
```
- 空タプル
```bash
t = ()
```
- tuple() コンストラクタ（イテラブルから生成）
```bash
t = tuple([1, 2, 3])   # リストから
t = tuple("ABC")       # 文字列から → ('A', 'B', 'C')
```
- 結合（+ 演算子）で新しいタプルを作成
```bash
t = (1, 2) + (3, 4)    # (1, 2, 3, 4)
```
- タプルは イミュータブル（変更不可） なデータ型であり、要素の追加・削除・代入はできない
- 読み取り専用のデータ保持に向いており、後で書き換えられたくない値を安全に管理できる
- リストとの違いは「可変か不変か」であり、タプルは固定されたデータを扱うのに適している
- 1要素タプルはカンマ必須であり、(1,) はタプルだが (1) は整数になる
- 空タプルは () で表現できる
- タプルの結合は + 演算子で可能であり、新しいタプルが生成される
- スライスやインデックス参照はリストと同様に利用できる
- count() や index() メソッドを使って要素の出現回数や位置を調べられる
- タプルの中に リストなどのミュータブルな要素を含めると、その部分は変更可能だが、タプル自体の構造は変えられない
- 辞書のキーや集合の要素として利用できる（イミュータブルなのでハッシュ可能）
