## 📝 VSCode デバッガ使い方

1. デバッグ開始
- .py ファイルを開く
- F5 または ▶ ボタンで開始
- 初回は launch.json が自動生成される（Python Debugger → Pythonファイル）

2. ブレークポイント
- 行番号の左をクリック → 赤丸が付く
- その行で処理が一時停止
- 条件付きブレークポイントも設定可能

3. ステップ実行
- F10：ステップオーバー（次の行へ）
- F11：ステップイン（関数の中へ）
- Shift+F11：ステップアウト（関数から抜ける）
- F5：続行（次のブレークポイントまで）

4. 変数確認
- 停止中は「変数」ペインで値を確認
- 「ウォッチ式」に変数や式を追加可能
- デバッグコンソールで式を直接評価できる

5. VSCode デバッガ ショートカット
- F5 : デバッグ開始（次のブレークポイントまで実行）
- F10 : ステップオーバー（関数呼び出しを飛ばして次の行へ）
- F11 : ステップイン（関数の中へ入る）
- Shift + F11 : ステップアウト（関数から抜けて外側へ戻る）
- 行番号左クリック : ブレークポイントの設定／解除
- Shift + F5 : デバッグの停止
- Ctrl + Shift + F5 : デバッグの再起動

6. VSCode デバッガ 各欄
- 変数ペイン → 全体を一覧で確認
- ウォッチ式 → 特定の変数や式を監視
- デバッグコンソール → 式を入力して評価
- ホバー表示 → コード上で即座に値を確認
